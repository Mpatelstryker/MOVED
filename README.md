# MOVED Autonomous Hospital Bed

A modern, interactive web presentation for the MOVED autonomous hospital bed system - featuring advanced safety features, intelligent navigation, and comprehensive healthcare benefits.

## Features

- **Autonomous Navigation** - Self-driving capability with advanced sensor technology
- **Advanced Fall Prevention** - Multi-layered safety systems
- **Real-Time Patient Monitoring** - Integrated vital signs monitoring
- **Infection Control** - UV-C sanitization and antimicrobial surfaces
- **Cloud-Based Integration** - EHR and IoT ecosystem connectivity

## Accessing the Page

1. Open `MOVED_Hospital_Bed.html` in your web browser
2. Enter one of the following passwords to unlock features:
   - `admin` - Full access to all sections
   - `demo` - Limited access (About section only)
   - `hide` - Minimal access
   - `spanish` - Enable Spanish language with minimal access

## Local Usage

Simply open the HTML file directly in your browser. No server required.

## GitHub Pages Setup

To deploy this to GitHub Pages:

1. Create a new GitHub repository (e.g., `hospital-bed`)
2. Push this code to the repository
3. Enable GitHub Pages in repository settings (Settings → Pages → Deploy from main branch)
4. Your site will be live at: `https://yourusername.github.io/hospital-bed`
